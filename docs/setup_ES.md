# Como levantar el entorno y correr el proyecto de forma local

## Fronend

### Requisitos

- `node` y `npm`

### Instalacion

```
    # ir al directorio client
    cd client

    # instalar dependenciaas
    npm install
```

### configuracion
```
    # en desarrollo, utilizar el hot reload
    npm run dev

    # al finalizar el desarrollo contruir
    npm run build
```
